Troubleshooting

If the database is offline

  1. Open config.php and update the MySQL host, database name, username, and password.
  2. Import sql/schema.sql into phpMyAdmin or the MySQL command line.
  3. Refresh the monitor page.

If file uploads fail

Make sure the uploads folder exists and is writable on the server.

If the theme does not change

Confirm that the site_settings table exists and contains the active_theme record. Then clear your browser cache.

If pages show missing styles

Confirm that assets/css/style.css and assets/js/app.js were uploaded in the same folder structure as the PHP files.

Troubleshooting guide complete.
