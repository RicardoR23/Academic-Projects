Admin Guide

Dashboard

The admin dashboard summarizes products, users, orders, requests, and monitoring checks.

Editing records

  1. Open Products.
  2. Click Edit on an existing item or Add new product.
  3. Update the name, price, options, and stock.
  4. Save to write the changes back to MySQL.

User administration

The Users page lets admins disable or re-enable non-admin accounts. Disabled users can no longer sign in.

Switching templates

The Settings page updates the active_theme site setting to classic, neon, or autumn, changing the look of every page.

Support responses

Admins can review support tickets, update status, and save a response in the Requests page.
