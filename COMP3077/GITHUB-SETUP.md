# GitHub Setup

To fully satisfy the repository requirement, push this folder to GitHub.

## Suggested steps

```bash
git init
git add .
git commit -m "Initial CreatorKart Gear project"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/creatorkart-gear.git
git push -u origin main
```

## Suggested branch structure
- `main` - stable deployment branch
- `dev` - active changes
- `feature/theme-refresh`
- `feature/admin-polish`
