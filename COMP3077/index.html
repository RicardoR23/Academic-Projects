CreatorKart Gear Help Wiki

This static help wiki explains how the site works for shoppers, admins, and anyone installing or updating the project.

What is included

  • Storefront catalog and product pages
  • User registration, login, profile, cart, checkout, and order history
  • Admin dashboard for products, users, requests, and themes
  • Monitoring page, media, SEO tags, and responsive navigation

Quick start for users

  1. Open the home page and browse the catalog.
  2. Create an account or log in.
  3. Visit a product page, choose two options, and add the item to cart.
  4. Go to checkout and place an order.
  5. Use the support page if you need help.

Quick start for admins

  1. Log in with the seeded admin account.
  2. Open the Admin Dashboard from the main menu.
  3. Use Products to edit records or add new ones.
  4. Use Users to disable accounts.
  5. Use Settings to switch among the three site templates.
